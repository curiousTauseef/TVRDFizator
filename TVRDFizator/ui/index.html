1. Introduction

Here we present the tool "TV Metadata Generator", developed by Eurecom under the scope of the European Proyect LinkedTV FP7. In a nutshell, this application takes a video and its corresponding metadata files, available in different formats, and produces a RDF representation of the whole information. The knowledge graph obtained as result is suitable to be interlinked with other data in the Web, and allow the execution of complex queries that can bring the viewers a new way of enjoy Television.

Documentation

The serialization of television content metadata into RDF is a complex process in which various well-known ontologies are involved (like The Open Annotation Core Data Model, the Ontology for Media Resources, or the Programmes Ontology). For extra information about this conversion, consult the following document:

LinkedTV Deriverable D2.2

LinkedTV Ontology

The LinkedTV model aims to make TV content and Web information seamlessly interconnected. This ontology defines a list of classes that can be relevant in the vast domain of television content, like for example Chapters, Scenes, Concepts, Objects... and allows to link them with information in external datasets.

Select Metadata Files

3. Choose Metadata files

In this section it is possible to specify the different files where it is possible to find information about the previously selected video. Those files contain mainly Legacy data, Analysis Results, Subtitles, and Entities, and they are available in different formats. The most important ones are:

Considered Formats

EXMARaLDA

It stores the Analysis Results obtained after the execution of different processes over the video. They include shot segmentation, scene segmentation, concept detection, automatic speech recognition, between others.

TV-Anytime

It stores the Legacy Metadata information provided by the Broadcasters, including for example the title, time and date of broadcasting, description, channel, and others.

SRT Subtitles

It is used for representing the subtitles of the selected video.

NERD Output

This format is a JSON serialization used for storing the Named Entities extracted by NERD from the video subtitles.
